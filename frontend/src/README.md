# Simple Todo List

A simple todo list application built with React, TypeScript, and Vite.

## Features

* Add, remove, and edit todo items
* Filter todo items by text

## Getting Started

### Prerequisites

* Node.js (version 14 or higher)
* npm (version 6 or higher)
* TypeScript (version 4 or higher)

### Installation

1. Clone the repository: `git clone https://github.com/justfrenzy22/srtlwf.git`
2. Install dependencies: `npm install`
3. Install TypeScript: `npm install typescript --save-dev`
4. Install Vite: `npm install vite --save-dev`

### Running the Application

1. Start the development server: `npm run dev`
2. Open your web browser and navigate to `http://localhost:3000`

### Building the Application

1. Build the application: `npm run build`
2. The built application will be stored in the `dist` directory

### Tutorial

1. Open the `src/App.tsx` file and add a new todo item by calling the `addTodo` function.
2. Filter todo items by text by calling the `filterTodos` function.
3. Edit a todo item by calling the `editTodo` function.
4. Remove a todo item by calling the `removeTodo` function.
